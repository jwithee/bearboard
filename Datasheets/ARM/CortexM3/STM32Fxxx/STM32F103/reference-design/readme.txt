This zip contains the STM3210B-EVAL board (MB525 & MB542) manufacturing database 
MB525 => Mother board version B-02
MB542 => LCD Daughter board version B-00

------------------------------------------------------------------
List of directories
------------------------------------------------------------------
MB525\assembly\	: MB525 Bill of material and assembly information files
MB525\pcb\gerber : MB525 PCB manufacturing files in Gerber format
MB525\schematics\ : MB525 Board rev B, Schematic rev. 1 and the associated PCB layout files in Protel version 2003 format,
                    as well as a PDF file with all schematic sheets. 
MB542\assembly\	: MB542 Bill of material and assembly information files
MB542\pcb\gerber : MB542 PCB manufacturing files in Gerber format
MB542\schematics\ : MB542 Board rev B, Schematic rev. 2 and the associated PCB layout files in Protel version 2003 format,
                    as well as a PDF file with all schematic sheets. 

------------------------------------------------------------------

For further documentation and downloads for this and other STMicroelectronic's microcontroller development tools, please refer to www.st.com/mcu.
