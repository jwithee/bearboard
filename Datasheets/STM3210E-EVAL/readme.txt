This zip contains the STM3210E-EVAL board (MB672 & MB694) manufacturing database 
MB672 => Mother board version D-00
MB694 => LCD Daughter board version A-00

------------------------------------------------------------------
List of directories
------------------------------------------------------------------
MB672\assembly\	: MB672 Bill of material and assembly information files
MB672\pcb\gerber : MB672 PCB manufacturing files in Gerber format
MB672\schematics\ : MB672 Board rev D, Schematic rev. 1 and the associated PCB layout files in Altium designer (Protel) version 6 format,
                    as well as a PDF file with all schematic sheets. 
MB694\assembly\	: MB694 Bill of material and assembly information files
MB694\pcb\gerber : MB694 PCB manufacturing files in Gerber format
MB694\schematics\ : MB694 Board rev A, Schematic rev.1 and the associated PCB layout files in Altium designer (Protel) version 6 format,
                    as well as a PDF file with all schematic sheets. 

------------------------------------------------------------------

For further documentation and downloads for this and other STMicroelectronic's microcontroller development tools, please refer to www.st.com/mcu.
